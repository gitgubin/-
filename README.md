本系统是一个基于Web的分布式服务器监控平台，提供主机管理和实时监控两大核心功能模块，帮助管理员高效管理多台远程服务器的运行状态。
主机管理模块
功能特点：
集中化管理：支持添加和管理多台远程服务器
SSH连接配置：通过IP地址、SSH用户名、密码和端口（默认22）建立安全连接
连接测试：内置SSH连接验证功能，确保配置正确性
批量操作：支持主机信息的添加、删除和批量管理
技术实现：
采用安全的SSH协议进行远程连接
支持多主机并发管理
提供实时连接状态检测

实时监控大屏
监控指标：
运行状态：实时显示服务器在线/离线状态）
CPU使用率：精确到小数点后一位
内存使用率：详细显示内存占用情况
实时更新：支持5秒自动刷新和手动即时刷新
可视化展示：
彩色状态标识（在线绿色/离线红色）
进度条式数据展示，直观易懂
时间戳记录，确保数据时效性
系统特性
用户体验：
响应式Web界面，支持多设备访问
简洁直观的操作流程，降低使用门槛
实时数据反馈，操作结果即时可见
技术架构：
基于Docker容器化部署，确保环境一致性
采用轻量级设计，资源占用低
支持扩展和定制化开发

应用场景
本系统特别适用于：
运维团队：集中监控多台服务器运行状态
开发环境：实时掌握测试服务器资源使用情况
教育机构：服务器管理教学和实践平台
中小企业：低成本构建内部监控系统
系统通过简洁的界面和强大的功能，为用户提供了专业级的服务器监控解决方案，有效提升了运维效率和管理便捷性。
